# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT open a public issue.**
2. Email [408736066@qq.com](mailto:408736066@qq.com) with details of the vulnerability.
3. Include steps to reproduce, impact assessment, and any suggested fixes if possible.

You should expect an initial response within 72 hours. We will work with you to understand and address the issue before any public disclosure.
